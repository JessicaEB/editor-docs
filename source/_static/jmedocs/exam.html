This chapter describes all the properties associated with an exam.

Navigation

It is possible to control how and when students are allowed to move between questions: for example, whether they are allowed to return to a question once they have completed it, whether they can jump between questions at will, or whether they must complete a question before moving on to another one.

PropertyDescriptionDefault value
allowregen Allow the student to re-randomise questions? false
reverse Allow the student to move back to the previous question? true
browse Allow the student to jump to any question at any time? true
showfrontpage Show an introduction page before starting the exam? true
